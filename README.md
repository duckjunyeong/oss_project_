🎯 **게임 방법:**

- **Paddle**을 좌우로 움직여 ↕️ 튕겨져 나오는 공을 받아치세요.
- 공이 화면 아래로 떨어지지 않도록 ⬇️ 주의하세요!
- 모든 벽돌을 💥 깨면 스테이지를 클리어하고 다음 단계로 넘어갑니다.
- 벽돌을 맞출 때마다 ✨ 포인트를 획득합니다.

🕹️ **조작법:**

- **← → (좌우 화살표):** Paddle 이동

- 다양한 아이템을 획득하여 🚀 게임을 유리하게 이끌어갈 수 있습니다.
